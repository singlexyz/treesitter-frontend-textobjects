# nvim-treesitter-frontend-textobjects

## Install

```
Plug 'nvim-treesitter/nvim-treesitter', {'do': ':TSUpdate'}
Plug 'nvim-treesitter/nvim-treesitter-textobjects'
Plug 'singlexyz/treesitter-frontend-textobjects' 
```
